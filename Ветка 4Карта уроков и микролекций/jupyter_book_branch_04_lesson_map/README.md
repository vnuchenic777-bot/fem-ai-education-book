# Ветка 4. Карта уроков и микролекций

Папка содержит Jupyter Book-совместимую методическую главу по ветке проектирования карты уроков и микролекций для курса ДПО:

> ИИ-инструменты в преподавательской, научной и повседневной работе: от понимания принципов до практического применения

## Состав папки

- `branch_04_lesson_map.md` — основная глава в формате MyST Markdown для Jupyter Book.
- `data/branch_04_lesson_map.yaml` — структурированная версия всех таблиц, перечней, уроков, микролекций, критериев и матриц.
- `data/branch_04_lesson_map.json` — машинно-читаемая версия для инженерных сред и автоматической обработки.
- `toc_fragment.yml` — фрагмент для подключения главы к `_toc.yml`.
- `README.md` — описание назначения папки.

## Назначение

Глава предназначена для включения в будущую Jupyter Book-книгу проекта. Она описывает методическую модель перехода от структуры курса ДПО к платформенным сущностям: `CourseTemplate`, `CourseModule`, `Lesson`, `CourseResource`, `AssessmentItem`, `LessonProgress`.

## Как подключить

Скопируйте папку в каталог книги и добавьте содержимое `toc_fragment.yml` в общий `_toc.yml`.

## Что требует внешнего согласования

Глава не утверждает нормативные требования. Порядок реализации ДПО, юридический статус документа об обучении, финальные правовые формулировки, правила хранения персональных данных и шаблон сертификата должны быть согласованы с образовательной организацией.
