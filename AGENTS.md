# Правила для AI-агентов

Этот репозиторий является исходником Jupyter Book-книги проекта «Проектирование цифровой образовательной экосистемы факультета».

## Обязательные правила

- Не конвертировать книгу в Word или PDF как основной формат.
- Не писать HTML вручную: HTML должен генерироваться через `jupyter-book build .`.
- Сохранять `.ipynb`, `.yaml`, `.yml`, `.json` и не превращать структурированные данные в обычный текст.
- Проверять сборку командой `jupyter-book build .`.
- Не удалять юридические дисклеймеры и предупреждения о необходимости согласования с образовательной организацией.
- Не менять названия и коды компетенций ПК-ИИ-1 — ПК-ИИ-10.
- Не придумывать нормативные ссылки, даты приказов, локальные акты или юридические утверждения.
- При изменении структуры книги обновлять `_toc.yml`.
- При добавлении главы добавлять README или краткое описание источника, а при необходимости — структурированные данные.
- После изменений запускать `python scripts/validate_data.py`, `python scripts/check_book.py` и `jupyter-book build .`, затем фиксировать результат в статусе публикации.

## Рабочий принцип

Публичные главы находятся в `book/`. Машинно-читаемые источники и копии веточных данных находятся в `data/branches/`. Изображения и диаграммы хранятся в `assets/` или рядом с соответствующей главой.
