# Модель хранения

## Назначение

Раздел должен описать модель хранения учебных материалов, файлов заданий, диаграмм, выданных документов и журналов аудита.

## Статус

Черновик.

## Что нужно перенести из веток

- Из ветки 10: рекомендацию хранить PDF-документ через StorageObject, а не внутри записи IssuedCertificate.
- Из ветки 5: хранение попыток отправки и результатов проверки.
- Из ветки 11: карты загрузки контента и материалов курса.

## Важное ограничение

Порядок хранения персональных данных, учебных работ, видеозаписей и документов об обучении требует отдельной проверки по официальным источникам и внутренним документам образовательной организации.
