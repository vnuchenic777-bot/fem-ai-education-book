# PWA-стратегия

## Назначение

Раздел должен описать пользовательскую стратегию Progressive Web App для образовательной платформы: прохождение курса, работа с уроками, сдача заданий, проверка, статус прогресса и доступ к документу об обучении.

## Статус

Черновик.

## Что нужно перенести из веток

- Из ветки 4: карта уроков и микролекций.
- Из ветки 5: пользовательский маршрут сдачи и проверки работ.
- Из ветки 10: маршрут просмотра документа и публичной QR-проверки.

## Открытые вопросы

- Какие пользовательские роли входят в первый MVP.
- Какие функции должны работать офлайн.
- Какие данные нельзя кэшировать на клиенте.
