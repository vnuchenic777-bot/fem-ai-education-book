# Библиография

## Назначение

Раздел фиксирует начальный список источников, на которые опирается публичная версия книги. Библиография будет расширяться по мере подготовки статей, уточнения нормативной рамки, развития архитектуры платформы и добавления эмпирических материалов.

## Нормативная рамка

- Федеральный закон № 273-ФЗ «Об образовании в Российской Федерации» используется как базовая рамка для обсуждения образовательных программ, документов о квалификации и документов об обучении {cite}`russian_education_law_273fz`.
- Приказ Минобрнауки России от 24.03.2025 № 266 используется как актуальная рамка для организации образовательной деятельности по дополнительным профессиональным программам {cite}`minobrnauki_order_266_2025_dpo`.
- Федеральный закон № 152-ФЗ «О персональных данных» используется как рамка для обсуждения публичной QR-проверки, обезличивания и границ раскрытия сведений о слушателях {cite}`personal_data_law_152fz`.

## ИИ в образовании

- Руководство UNESCO по генеративному ИИ в образовании и исследованиях используется для человекоцентричной, этической и институционально ответственной рамки применения ИИ {cite}`unesco_genai_education_2023`.
- Рамка компетенций UNESCO для преподавателей используется как внешний ориентир при описании профессиональных компетенций работы с ИИ {cite}`unesco_ai_competency_teachers_2024`.
- Принципы OECD по ИИ используются как общий ориентир доверенного, безопасного, прозрачного и подотчетного применения ИИ-систем {cite}`oecd_ai_principles_2024`.

## Техническая публикация

- Документация Jupyter Book используется для настройки `_config.yml`, `_toc.yml`, сборки и публикации книги как автоматически генерируемого HTML-ресурса {cite}`jupyterbook_config_v1,jupyterbook_toc_v1`.
