# Сертификат или удостоверение

Эта папка содержит материалы раздела **«Сертификат или удостоверение»** для Jupyter Book-книги:

**Проектирование цифровой образовательной экосистемы факультета: методология, архитектура и реализация курса ДПО по ИИ-инструментам**

Материалы подготовлены не как Word, PDF или HTML, а как исходники для открытой научно-методической книги на GitHub Pages.

## Назначение

Раздел описывает модель документа об обучении для курса ДПО:

> ИИ-инструменты в преподавательской, научной и повседневной работе: от понимания принципов до практического применения

Документ об обучении должен подтверждать прохождение курса, освоение 72 академических часов, итоговый статус и компетенции **ПК-ИИ-1 — ПК-ИИ-10**.

## Важное ограничение

Этот раздел не утверждает юридический статус документа. Порядок выдачи, подписи, регистрации, хранения и учета должен быть согласован с образовательной организацией и ответственными специалистами института.
