# Итоговый проект: рабочая тетрадь

Файл `final_project_workbook.ipynb` содержит адаптированную Jupyter-версию исследования ветки 8: итоговый проект курса ДПО.

## Назначение

Рабочая тетрадь дополняет главу `final_project_model.md` и переводит часть материалов итогового проекта в воспроизводимый табличный формат. Ее можно использовать для проверки структуры проекта, рубрики оценивания, рисков, API-контура и чек-листов.

## Состав notebook

- назначение итогового проекта;
- структура итогового проекта;
- шаблон для слушателя;
- инструкция по выполнению;
- требования к заданиям с обязательным использованием ИИ;
- рубрика оценивания;
- критерии зачета и доработки;
- типичные ошибки;
- риски и меры снижения;
- формат асинхронной видео-защиты;
- связь с образовательной платформой;
- техническое задание на реализацию;
- API-контур;
- чек-листы;
- экспорт таблиц в CSV и Markdown.

## Статус

Черновик, подключенный к Jupyter Book как воспроизводимая глава ветки 8.

```{important}
Notebook не заменяет юридическую экспертизу. Вопросы официального статуса документа об обучении, персональных данных, подписи, регистрационного номера и локальных актов образовательной организации должны проверяться отдельно.
```
