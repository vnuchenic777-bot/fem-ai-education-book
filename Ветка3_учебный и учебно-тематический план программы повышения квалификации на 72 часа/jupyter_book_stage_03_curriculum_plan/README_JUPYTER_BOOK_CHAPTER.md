# Jupyter Book-глава: Ветка 3. Учебный и учебно-тематический план

## Название ноутбука

`stage_03_curriculum_plan_chapter.ipynb`

## Назначение

Ноутбук представляет отдельную главу будущей Jupyter Book-книги проекта. Он работает строго в рамках ветки 3: учебный и учебно-тематический план программы повышения квалификации на 72 часа.

## Файлы данных

- `data/stage_03_curriculum_plan.yaml` — человекочитаемая версия данных.
- `data/stage_03_curriculum_plan.json` — машинно-читаемая версия данных.
- `data/stage_03_modules.csv` — таблица модулей.
- `data/stage_03_topics.csv` — таблица тем.
- `data/stage_03_calendar.csv` — календарный график.
- `data/stage_03_platform_requirements.csv` — связь с объектами платформы.
- `data/stage_03_risks.csv` — риски этапа.
- `data/stage_03_acceptance_criteria.csv` — критерии приемки.

```yaml
  - file: stage_03_curriculum_plan_chapter
    title: Ветка 3. Учебный и учебно-тематический план
```

Если ноутбук лежит в подпапке, например `chapters/stage_03/`, подключение будет таким:

```yaml
format: jb-book
root: intro
chapters:
  - file: chapters/stage_03/stage_03_curriculum_plan_chapter
    title: Ветка 3. Учебный и учебно-тематический план
```

## Научно-публикационный потенциал

Ноутбук может стать основой:

1. методической статьи о проектировании ДПО-программы для преподавателей по применению ИИ-инструментов;
2. доклада о компетентностной структуре курса и учебно-тематическом планировании;
3. главы исследовательского журнала проекта, где методическое решение представлено как проверяемая структура данных;
4. эмпирической статьи после пилотного запуска, если добавить данные о прохождении, результатах тестов, качестве артефактов и типичных ошибках слушателей.

Главная научная ценность формата — воспроизводимость: таблицы, графы, проверки сумм часов и модели связей строятся кодом, а не вручную в текстовом редакторе.
