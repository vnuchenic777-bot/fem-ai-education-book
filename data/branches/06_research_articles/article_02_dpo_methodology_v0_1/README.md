Этот каталог фиксирует редакционный пакет второй статьи научно-публикационного контура.


## Состав

## Статус

Рабочая публикационная редакция v0.2. Статья описывает не локальный отчёт о конкретном факультете, а универсализируемую методологию проектирования программ ДПО по ИИ-инструментам для преподавателей. Авторская программа на 72 часа используется как апробационная конфигурация модели.

Версия v0.2 добавляет внешний обзор исследований по ДПО педагогов, цифровым компетенциям, международным рамкам компетенций, оцениванию результатов, отношению преподавателей к ИИ и барьерам внедрения ИИ в ДПО.
