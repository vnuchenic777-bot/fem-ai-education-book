# Ветка 5. Фонд оценочных средств

Эта папка содержит материалы исследовательско-методической ветки, посвящённой фонду оценочных средств для программы повышения квалификации:

> «ИИ-инструменты в преподавательской, научной и повседневной работе: от понимания принципов до практического применения».

Материалы подготовлены для включения в будущую публичную Jupyter Book-книгу о проектировании цифровой образовательной платформы ФЭМ СПБГТИ(ТУ), курса ДПО и методической модели внедрения ИИ в образовательный процесс.

## Назначение папки

Папка фиксирует модель контроля результатов обучения, подтверждения компетенций, проверки артефактов, тестирования, повторной сдачи, итогового проекта и асинхронной защиты.

## Состав файлов

- `main_chapter.md` — основная глава Jupyter Book.
- `notebook_01.ipynb` — вычислительная тетрадь для построения матрицы соответствия компетенций, модулей и форм контроля.
- `notebook_02.ipynb` — вычислительная тетрадь для визуализации жизненного цикла проверки и технической модели оценивания.
- `data/structured_data.yaml` — структурированные данные ветки для методической и инженерной работы.
- `data/structured_data.json` — те же данные в формате JSON.
- `assets/diagrams/` — папка для схем, которые строятся вычислительными тетрадями.
- `assets/images/` — папка для будущих изображений.
- `toc_fragment.yml` — фрагмент оглавления для подключения ветки к Jupyter Book.

## Как использовать

1. Подключить `main_chapter.md` в оглавление Jupyter Book.
2. Разместить папку ветки внутри общего каталога книги, например `branches/branch_05_fos/`.
3. Проверить пути в `toc_fragment.yml` и при необходимости заменить префикс.
4. Запустить вычислительные тетради для построения схем и проверочных таблиц.
5. Передать `data/structured_data.yaml` и `data/structured_data.json` инженерной команде как основу для переноса ФОС в платформу.

## Ограничения

Материал не утверждает юридический статус документа об обучении. Порядок выдачи сертификата или удостоверения, состав комиссии, итоговые пороги и работа с персональными данными требуют проверки по официальным источникам и внутренним документам образовательной организации.
