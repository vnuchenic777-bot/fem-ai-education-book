# Jupyter Book-глава: AI-mandatory задания

Папка содержит отдельную главу будущей Jupyter Book-книги проекта ФЭМ СПБГТИ(ТУ) «Образование».

## Назначение

Глава фиксирует техническое и методическое задание на разработку этапа «Задания с обязательным использованием ИИ студентами». Материалы предназначены для:

- методистов ДПО;
- преподавателей высшей школы;
- разработчиков образовательной платформы;
- научных редакторов и авторов будущих публикаций;
- инженерных сред, которым нужна структурированная версия таблиц, критериев и рубрик.

## Состав

```text
chapters/ai_mandatory_assignments_tz.md
  Основная Jupyter Book-совместимая Markdown-глава.

data/ai_mandatory_assignments_tz.yaml
  Структурированная версия таблиц, критериев, рубрик, примеров и открытых вопросов.

data/ai_mandatory_assignments_tz.json
  Машинно-читаемая версия для инженерной среды и дальнейшей автоматизации.

toc_fragment.yml
  Фрагмент подключения главы к _toc.yml.

README.md
  Краткое описание папки.
```

## Использование в Jupyter Book

1. Скопировать `chapters/ai_mandatory_assignments_tz.md` в папку `chapters/` вашей Jupyter Book-книги.
2. Скопировать папку `data/` в корень книги или в согласованную папку данных.
3. Добавить запись из `toc_fragment.yml` в основной `_toc.yml`.
4. Собрать книгу стандартным способом вашего проекта.

## Важное ограничение

Глава не утверждает нормативные требования. Все вопросы, связанные с локальными актами, персональными данными, официальным оцениванием и порядком включения задания в РПМ, требуют отдельного согласования с образовательной организацией.
